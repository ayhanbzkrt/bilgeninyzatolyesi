# Bilge'nin Yapay Zekâ Atölyesi  
## Çocuklar İçin Etkileşimli Yapay Zekâ Öğrenme Portalı

**Bilge'nin Yapay Zekâ Atölyesi**, çocukların yapay zekâ kavramlarını deneyerek, oynayarak ve keşfederek öğrenmeleri için hazırlanmış etkileşimli bir eğitim portalıdır. Portal; veri, algoritma, bilgisayarların öğrenmesi, karar verme, görsel tanıma, ses tanıma, güvenli yapay zekâ kullanımı ve dijital okuryazarlık gibi temel konuları çocuk dostu bir anlatımla sunar.

Bu proje, **AI'Han Academy** tarafından geliştirilen eğitsel, etkileşimli ve tarayıcı tabanlı bir öğrenme deneyimidir.

---

## 🇹🇷 Proje Hakkında

Bu portal, çocukların “Yapay Zekâ nedir?”, “Bilgisayarlar nasıl öğrenir?”, “Veri ne işe yarar?”, “Yapay zekâ doğru karar verir mi?” gibi sorulara uygulamalı etkinliklerle yanıt bulmasını amaçlar.

Portal içinde çocuklar:

- Verinin ne olduğunu keşfeder.
- Algoritma mantığını adım adım öğrenir.
- Bilgisayarların örneklerden nasıl öğrendiğini deneyimler.
- Yapay zekânın karar verme süreçlerini gözlemler.
- Görsel, ses, metin, çizim ve müzik odaklı yapay zekâ etkinlikleriyle etkileşime girer.
- Güvenli, bilinçli ve sorumlu yapay zekâ kullanımı hakkında farkındalık kazanır.
- Atölye ilerlemesini takip eder ve final rozeti deneyimi yaşar.

---

## 🇬🇧 About the Project

**Bilge's Artificial Intelligence Workshop** is an interactive learning portal designed to help children understand the fundamentals of artificial intelligence through exploration, play, and hands-on activities.

The portal introduces core AI concepts such as data, algorithms, machine learning, decision-making, computer vision, speech recognition, creative AI, and safe AI use in a child-friendly way.

Through the workshop, children can:

- Explore what data means.
- Learn algorithmic thinking step by step.
- Understand how computers learn from examples.
- Observe how AI systems make decisions.
- Interact with visual, audio, text, drawing, music, and storytelling activities.
- Develop awareness of safe, responsible, and ethical AI use.
- Track their workshop progress and complete the final badge experience.

---

## 🎯 Amaç / Purpose

Bu projenin temel amacı, çocukların yapay zekâ okuryazarlığını erken yaşta desteklemek ve yapay zekâyı yalnızca teknik bir konu olarak değil; merak, sorgulama, yaratıcılık, güvenlik ve etik sorumluluk bağlamında anlamalarını sağlamaktır.

The main purpose of this project is to support children's AI literacy at an early age and help them understand artificial intelligence not only as a technical subject, but also through curiosity, questioning, creativity, safety, and ethical responsibility.

## 🚀 Kullanım / Usage

Projeyi çalıştırmak için:

1. Bu repoyu indirin veya klonlayın.
2. `index.html` dosyasını bir tarayıcıda açın.
3. Atölye menüsünden istediğiniz modülü seçerek kullanmaya başlayın.

To use the project:

1. Download or clone this repository.
2. Open the `index.html` file in a web browser.
3. Select a module from the workshop menu and start exploring.

## 🌐 GitHub Pages ile Yayınlama / Publishing with GitHub Pages

Bu proje statik HTML yapısında olduğu için GitHub Pages üzerinde doğrudan yayınlanabilir.

1. Repository ayarlarına girin.
2. **Settings > Pages** bölümünü açın.
3. Branch olarak `main` veya `master` seçin.
4. Kök dizini `/root` olarak belirleyin.
5. Kaydedin ve GitHub Pages bağlantısının oluşmasını bekleyin.

Since this is a static HTML project, it can be published directly with GitHub Pages.
