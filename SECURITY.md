# SECURITY  
## Security Policy for Bilge'nin Yapay Zekâ Atölyesi

Bu belge, **Bilge'nin Yapay Zekâ Atölyesi** projesi için güvenlik, bildirim ve sorumlu kullanım ilkelerini açıklar.

This document explains the security, reporting, and responsible use principles for **Bilge's Artificial Intelligence Workshop**.

---

## Türkçe

### Desteklenen Sürüm

Bu proje tek dosyalı, tarayıcı tabanlı bir eğitim portalı olarak geliştirilmiştir.

Güvenlik değerlendirmeleri ve güncellemeler öncelikli olarak repository’de yer alan güncel sürüm için geçerlidir.

| Sürüm | Destek Durumu |
|---|---|
| Güncel `main` sürümü | Desteklenir |
| Eski veya değiştirilmiş kopyalar | Desteklenmez |

---

### Güvenlik Kapsamı

Bu proje:

- Çocuklara yönelik eğitimsel bir Yapay Zekâ okuryazarlığı portalıdır.
- Tarayıcı üzerinde çalışacak şekilde hazırlanmıştır.
- Ticari olmayan eğitimsel kullanım için tasarlanmıştır.
- Kullanıcıdan hassas kişisel veri toplamayı amaçlamaz.
- Sunucu taraflı veri işleme, kullanıcı hesabı, ödeme sistemi veya üyelik altyapısı içermez.

---

### Güvenlik Bildirimi

Projede aşağıdaki türde bir sorun fark ederseniz lütfen bildiriniz:

- Zararlı kod şüphesi
- Çocuk güvenliğini etkileyebilecek içerik veya yönlendirme
- Yanıltıcı, riskli veya uygunsuz Yapay Zekâ açıklaması
- Tarayıcı güvenliğiyle ilgili açık
- İzinsiz üçüncü taraf bağlantısı veya veri aktarımı
- Telif hakkı, marka veya içerik ihlali şüphesi

Bildirim yaparken mümkünse şu bilgileri ekleyiniz:

1. Sorunun kısa açıklaması  
2. Sorunun görüldüğü dosya veya bölüm  
3. Sorunu yeniden üretme adımları  
4. Ekran görüntüsü veya hata çıktısı  
5. Önerilen çözüm, varsa  

---

### Sorumlu Bildirim İlkesi

Güvenlik veya içerik sorunlarını bildirirken lütfen:

- Sorunu kamuya açık şekilde istismar etmeyiniz.
- Çocuk kullanıcıları etkileyebilecek içerikleri yaygınlaştırmayınız.
- Proje kodunu veya içeriğini kötüye kullanmayınız.
- Ticari amaçla çoğaltma, satma veya yeniden yayımlama yapmayınız.
- AI'Han Academy’nin telif ve kullanım koşullarına uyunuz.

---

### Çocuk Güvenliği ve Etik Kullanım

Bu proje çocuklara yönelik olduğu için aşağıdaki ilkelere özellikle önem verilir:

- Yaşa uygun dil ve içerik

### Supported Version

This project is developed as a single-file, browser-based educational portal.

### Security Scope

This project:

- Is an educational AI literacy portal for children.
- Is designed to run in the browser.
- Is intended for non-commercial educational use.
- Does not aim to collect sensitive personal data.
- Does not include server-side data processing, user accounts, payment systems, or membership infrastructure.

---

### Reporting a Security Issue

Please report any of the following issues if you notice them:

- Suspected malicious code
- Content or links that may affect child safety
- Misleading, risky, or inappropriate AI explanations
- Browser-related security vulnerabilities
- Unauthorized third-party links or data transfer
- Suspected copyright, trademark, or content infringement

When reporting an issue, please include:

1. A brief description of the issue  
2. The file or section where the issue appears  
3. Steps to reproduce the issue  
4. Screenshot or error output, if available  
5. Suggested fix, if any  

---

### Responsible Disclosure

When reporting security or content issues, please:

- Do not publicly exploit the issue.
- Do not spread content that may affect child users.
- Do not misuse the project code or content.
- Do not reproduce, sell, or republish the project for commercial purposes.
- Follow AI'Han Academy’s copyright and usage terms.

---

### Child Safety and Ethical Use

Because this project is designed for children, the following principles are especially important:

- Age-appropriate language and content
- Safe use of Artificial Intelligence
- Awareness of not sharing personal data
- Critical thinking
- Information verification
- Awareness against misinformation
- Emphasis that AI-generated outputs should always be checked
